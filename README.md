# Sistema-de-gestion-de-envios-logiExpress
Taller#3

Descripcion:LogiExpress es un sistema de gestión de envíos desarrollado en Java bajo el paradigma de Programación Orientada a Objetos (POO).
El proyecto aplica los conceptos de herencia, polimorfismo y principios SOLID, permitiendo gestionar diferentes tipos de envíos (terrestres, aéreos y
marítimos) de manera modular y extensible.
